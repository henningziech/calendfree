# Welcome to Calendfree

Calendfree is a self-hosted round-robin scheduling platform that integrates with Google Workspace. It lets teams share booking pages, distribute appointments automatically, and manage availability — all backed by your existing Google Calendar.

Think of it as a Calendly replacement you can run on your own infrastructure, with full control over your data.

## Who is it for?

- **Teams** that need to distribute inbound meetings across members (sales, support, consulting)
- **Organizations** that want a self-hosted scheduling solution with Google Workspace integration
- **Developers** looking for a programmable scheduling backend with a REST API

## Key Features

### Event Types

Create personal, team (round-robin), or group event types with configurable duration, buffer times, minimum notice, and booking windows. Each event type gets a public booking URL.

### Round-Robin Scheduling

Distribute appointments across team members using three strategies: **sequential** (strict rotation), **least-busy** (fewest upcoming bookings), or **weighted** (custom distribution ratios).

### Google Calendar Integration

Slots are calculated from real-time Google Calendar availability. Bookings automatically create Google Calendar events with optional Google Meet links.

### Availability Management

Set weekly schedules per user, define date-specific overrides, block holidays by country, and set daily/weekly booking limits.

### Routing Forms

Pre-qualify visitors with custom forms that route them to the right event type based on their answers.

### Branding

Customize booking pages with your company logo, colors, and footer text.

### Embed Widget

Add scheduling to any website with a single script tag.

### API Keys

Programmatic access to the full API for building integrations and automations.

### Analytics

Track booking statistics and team performance from the admin dashboard.
